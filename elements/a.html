<a>

Ancla

Definición

La etiqueta <a> crea un hipervínculo o un ancla. Un hipervínculo se utiliza para vincular a otro documento. Un ancla se utiliza para vincular a otro punto dentro del mismo documento.

¿Cómo recordar?

El ancla de HTML es como el ancla de un barco. El ancla de un barco mantiene el barco en su posición. Un ancla de HTML mantiene la posición dentro de la misma página web o la ubicación de otra página web.

Anchor

Definition

The <a> tag creates a hyperlink or an anchor. A hyperlink is used to link to another web page. An anchor is used to link to a specific point within the same web page.

How to Remember

The HTML anchor works like a boat anchor. A boat anchor holds the boat in its location. An HTML anchor holds a location within the same web page or the location of another web page.

    
      <p>This is a hyperlink to Google:
        <a href=”http://www.google.com”>Google</a>
      </p>

      <p>This is a link to another place on the same page:
        <a href="#credits">Credits</a>
      </p>
