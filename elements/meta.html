<meta>

English Base: Metadata
Spanish translation/Traducción en Español: Metadatos

Definición

La etiqueta <meta> se utiliza para representar información que no se puede expresar con <base>, <link>, <script>, <style>, y <title>. Algunos tipos de información que representa son atributos como nombre y contenido.

Definition

The <meta> element is used to represent information that cannot be expressed by using <base>, <link>, <script>, <style>, and <title>. Some types of information this represents are attributes like name and content.

¿Cómo recordar?

Los metadatos son como un metalenguaje en que un metalenguaje es un lenguaje sobre lenguaje y los metadatos son datos sobre los datos. Detalla exactamente el tipo de lenguaje está permitido.

How to Remember

Metadata is like a metalanguage in that metalanguage is language about language and metadata is data about data. It spells out exactly what type of language is allowed.

    
      <html>
        <meta content="author" name="John Smith">
      </html>
