<base>

Traducción en Español:

Base

English base:

Base

Definición

La etiqueta <base> se utiliza para establecer el URL base para todos los URL relativos dentro del documento. Sólo puede haber una etiqueta <base> en el documento.

Definition

The <base> tag is used to establish what the base URL will be for all relative URL's within the document. There can be only one <base> tag in a document.

¿Cómo recordar?

Una casa está construida sobre una base: los cimientos. Una casa sólo tiene una dirección, no importa cuántas habitaciones tiene. Una página web sólo tiene una <base> (cimientos) en las que se basan sus páginas (habitaciones).

How to Remember

A house is built on a base foundation. A house only has one address, no matter how many rooms it has. A web page only has one <base> (foundation) that its pages (rooms) are based on.

    
      <html>
        <head>
          <base href="http://www.adorable.io">
        </head>
      </html>
