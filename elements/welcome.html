Introduction to LMHT

Welcome to LMHT. This is an open source code repository to help Spanish speakers understand HTML tags. LMHT was originally started as a YWebCA project by Team Open Brace in the Summer of 2015. Adorable and other contributors are working towards expanding this project into a valuable learning tool for bilingual programmers.

¿Cómo utilizar LMHT?

Empiece a escribir el nombre de una etiqueta HTML el el cuadro de búsqueda arriba y elija una etiqueta en la lista. También puede hacer clic al botón a la derecha del cuadro de búsqueda para ver las etiquetas HTML disponibles. Recuerde que este es un trabajo en progreso, por lo que todavía no tenemos todas las etiquetas. Una vez que haya hecho su selecció, la siguiente informació se dará en Español e Inglé: definiciones, grabaciones de las pronunciaciones y definiciones, pistas para recordar, y ejemplos de código

How to Use LMHT

Begin typing the name of a HTML tag in the search box above, then choose a tag in the list. You may also click the button on the right of the search box to see all the available HTML tags. Remember this is a work in progress, so we do not have all tags yet. Once you have made your selection, the following information will be given in English and Spanish: definitions, audio recordings of the pronunciations and definitions, hints for how to remember, and code samples.

¿Cómo Contribuir?

How to Contribute to LMHT?

If you have any suggestions, you can contribute to the project by going to our Github site. Please read the CONTRIBUTING.md before submitting an issue. If you're not familiar with using GitHub, email us with any feedback or questions. We will try our best to get back to you within a couple of days.
