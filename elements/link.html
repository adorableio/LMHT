<link>

English Base: Link
Spanish translation/Traducción en Español: Enlace

Definición

La etiqueta <link> establece una conexión a un recurso fuera de la página web. Normalmente se usa para vincular una hoja de estilos al documento. Esta etiqueta suele contener el atributo 'href', que puede indicar al navegador donde se encuentra una hoja de estilos, por ejemplo.

Definition

The <link> tag establishes a connection to a resource outside of the website. It is mostly used to link a stylesheet to the document. This tag usually contains the 'href' attribute, which may tell the browser where a stylesheet is found, for instance.

¿Cómo recordar?

Una cadena se hace con enlaces que conecta una cosa a otra.

How to Remember

A chain is made with links, which attaches one item to another.
