<body>

English base: Body
Spanish translation/Traducción en Español: Cuerpo

Definición

La etiqueta <body> contiene el cuerpo del documento. Contiene todo el contenido visible de una página. Los contenidos del <body> pueden contener texto, fotos, hiperenlaces, tabla, listas, etc.

Definition

The <body> tag makes up the body of a document. It contains all the visible content of a page. The contents of <body> might include text, images, hyperlinks, tables, lists, etc.

¿Cómo recordar?

Las secciones de una página HTML son lo mismo como que las secciones de un documento escrito. La parte de medio de un documento es conocido como el cuerpo (body en inglés), al igual como el parte de medio de una página de HTML es conocido como el <body>. El cuerpo (<body>) contiene el contenido principal de una página HTML o un documento.

How to Remember

The main sections of an HTML page are the same as the sections of a written document. The middle part of a document is known as the body, just like the middle part of an HTML page is known as the body. The body contains the main content.

    
      <body>
        <p>The content of a document includes</p>

            <li>tables, and</li>
            <li>lists</li>
          </ul>
      </body>
