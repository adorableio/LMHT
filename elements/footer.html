<footer>

English base: Footer
Spanish translation/Traducción en Español: Pie de Página

Definición

La etiqueta <footer> es un elemento de bloque que crea el pie de página de una sección, artículo, o página. Incluso contiene información del contenido como información de contacto, derechos de autoría, o información general. El footer es un elemento semántico usado en lugar de <div> para ayudar a buscadores encontrar información relevante a la página.

Definition

The <footer> tag is a block element that creates the footer at the bottom of a section, article, or page. It also contains informational content, like contact info, copyright info, or general site info. The footer is a semantic element used in place of a <div> to help search engines find relevant information about your page.

¿Cómo recordar?

Las secciones de una página HTML son lo mismo que las secciones de un documento escrito. La parte inferior de un documento es conocido como el pie de página (“footer” en inglés), al igual que la parte inferior de una página HTML es conocido como el <footer>.

How to Remember

The main sections of an HTML page are the same as the sections of a written document. The bottom part of a document is known as the footer, just like the bottom part of an HTML page is known as the footer.
